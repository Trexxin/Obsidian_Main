#### Changing verbs forms into ~ます and ～ません  
Dictionary Form →　Present, affirmative form →　Present, negative form
1. 聞く　→　聞きます　→　聞きません
2. 飲む　→　飲みます　→　飲みません
3. 話す　→　話します　→　話しません
4. 読む　→　読みます　→　読みません
5. 見る　→　見ます　→　見ません
6. する　→　すます　→　すません
7. 勉強する　→　勉強します　→　勉強しません
8. 行く　→　行きます　→　いきません
9. 帰る　→　帰ります　→　帰りません
10. 来る　→　来ます　→　来ません
11. おさる　→　あさります　→　あさりません
12. ねる　→　ねます　→　ねません
